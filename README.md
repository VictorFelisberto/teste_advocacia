# Silva Advocacia — Site Institucional

Projeto de um site institucional para escritório de advocacia, desenvolvido com apoio do Bolt e personalizado por mim para fins de estudo, portfólio e demonstração de capacidade de construção de interfaces modernas para negócios reais.

## Visão geral

O objetivo deste projeto foi criar uma landing page profissional para um escritório de advocacia, com foco em apresentação dos serviços, autoridade visual, navegação clara e facilidade de contato para potenciais clientes.

O site foi estruturado com seções estratégicas para um negócio jurídico, buscando transmitir credibilidade, organização e boa experiência de navegação.

## Demonstração

- **Repositório:** [teste_advocacia](https://github.com/VictorFelisberto/teste_advocacia)
- **Deploy:** [Acessar projeto online](https://silvaadvocacia.bolt.host)

## Funcionalidades

- Página inicial com apresentação do escritório
- Seção de áreas de atuação
- Bloco de experiência e indicadores institucionais
- Seção de notícias jurídicas
- Área de contato com informações e formulário
- Layout responsivo
- Navegação por seções
- Estrutura visual moderna para apresentação profissional

## Tecnologias utilizadas

- React
- TypeScript
- Vite
- Tailwind CSS
- Lucide React

## Estrutura do projeto

O projeto foi dividido em componentes para facilitar manutenção, reutilização e organização da interface.

Principais seções:
- Header
- Hero
- Áreas de Atuação
- Experiência
- Notícias
- Citação
- Contato
- Footer
